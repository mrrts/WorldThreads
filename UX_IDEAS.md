# UX Ideas (non-prescriptive)

The agent has wide latitude; these are guiding concepts only.

## Core surfaces

- Character Threads: two persistent conversations, iMessage-like feel.
- World Feed: timeline of compact “meanwhile…” world events.
- World Bible Editor: user defines setting/rules/tone/invariants.
- Character Editor: canon sheet + voice examples + boundaries.
- Scene View (Three.js): characters exist in a place; subtle animation and presence.

## Interaction ideas

- Switching characters feels like switching threads; the world feed continues.
- World events can be visible as small “cards” (click to view details/hook list).
- A “Since you last spoke…” one-liner can show in UI without costing tokens.
- World interactions (click object/location) can generate a “perception event” injected as a short system note for dialogue.

## Player controls

- Set world time/location (simple controls).
- Edit canon (with provenance).
- “Retcon last tick” (undo last world tick patch + event) as a safety valve.
- Export/import world.

## Tone

- Chat-first, with 3D as ambience and embodiment.
