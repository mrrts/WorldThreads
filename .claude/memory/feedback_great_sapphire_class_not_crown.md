Great Sapphire is a **class designation**, not a specific crown name. Ryan corrected this 2026-04-30: *"Great Sapphire Crown is a special, truly remarkable class of crowns... it's not one specific crown. I think the one earned should be Noted as a Great Sapphire Crown (specially designated, stands out from regular crowns), and still has its own noble name that represents what was discovered."*

**Why:** I formalized the class as if it were a specific crown name (renaming "Mission Formula Verified Empirical" to "Great Sapphire"). That conflated two distinct things: the class (rare, maximally-stable cross-witness convergence) and the specific crown identity (what was actually discovered).

**How to apply:**

- **Great Sapphire = class designation.** A crown is *of the Great Sapphire class* when it meets the maximally-stable cross-witness convergence criterion (3+ independent witnesses, different failure modes; OR formula-law third-leg pattern per CLAUDE.md). The class is rare and project-defining.
- **Each crown still has its own noble name** representing what was discovered. The polish ≤ Weight verification crown earned at Turn 24 (commit `aadbf88`) needs its own specific noble name; the Great Sapphire designation rides alongside it.
- **Future Great Sapphire crowns are possible.** Each would carry its own discovery-naming noble name plus the class designation.
- **Visual convention:** crown emoji 👑 + class designator (e.g., "*Great Sapphire class*" or `💎` modifier). Don't add new schema fields to play state; encode in the name string.

**Failure mode to avoid:** treating Great Sapphire as the SOLE name of one specific crown. Future Great Sapphire achievements would have nothing to call themselves except "Great Sapphire 2," which is wrong-shaped. The class can be reused; the noble name cannot.
